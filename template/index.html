Automated Roman Numeral Analysis

By Mike Cuevas, Aaron Keller, and David Zane


What Am I?

We will be writing a program to perform an automated Roman numeral harmonic analysis of an input chord progression signal. As an additional step, we will then convert that sequence of chords into their negative harmony equivalents. We are building on existing systems to give musicians a way to experiment with negative harmonies.

Why?

Roman numeral analysis is an extremely important tool in analyzing the functions chords serve in a piece of music. It allows musicians to understand why a composer decided to place certain chords where they did and can help improve performance. Creating an automated version of this would be an important tool for musicians. This tool could be modified one day to not only work for input chord progressions, but to be able to analyze input music. This has the potential to benefit many fields of machine music perception.

Additionally, one of the most recent contributions to the world of music theory in regards to functional harmony is the idea of negative harmony which can allow musicians to experiment with new colors. Writing a program that can perform a Roman numeral analysis and then allow the user to see what certain chords could sound like with their ‘negative’ equivalents would be a useful tool for musicians today.
